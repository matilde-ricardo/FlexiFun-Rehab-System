# FlexiFun: Sensorized Glove & Unity Game for Hand Rehabilitation

FlexiFun is an integrated biofeedback system designed to motivate patients during physiotherapy and allow therapists to monitor clinical evolution in real-time.

## Key Features
- **Sensorized Glove:** Collects hand movement data (hardware integration).
- **Unity Game:** A gamified experience that encourages patients to perform specific exercises.
- **Web Dashboard (PHP):** A platform for physiotherapists to track patient progress.
- **Database (MySQL):** Secure storage of session data and recovery metrics.

## Tech Stack
- **Game Engine:** Unity (C#)
- **Web:** PHP, HTML, CSS
- **Database:** MySQL
- **Hardware:** Sensorized Glove (Arduino/Microcontroller integration)

## Impact
This project bridges the gap between traditional physiotherapy and digital health, increasing patient engagement through gamification while providing objective data for clinical analysis.
